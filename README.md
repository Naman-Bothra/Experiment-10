#### Aim : Study of pandas library

#### Theory : 
Pandas is an open-source Python library used for data manipulation, analysis, and cleaning. It is especially popular in data science, machine learning, finance, and analytics.

Main Features of Pandas

1. Data Structures

Pandas provides two primary data structures:

Series → 1-dimensional labeled array (like a column in Excel)

DataFrame → 2-dimensional labeled table (like a spreadsheet or SQL table)

2. Data Cleaning

Handling missing values

Removing duplicates

Renaming columns

Filtering rows

3. Data Analysis

Statistical calculations (mean, median, sum, etc.)

Grouping data (groupby)

Sorting

Aggregations
